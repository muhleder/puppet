

Something like this in your VagrantFile

config.vm.share_folder "puppet-data", "/var/puppet", "~/workspace/puppet"

Then

vagrant ssh

and run the following script

#*************** START SCRIPT **********************
#!/bin/bash

# Set up some defaults from the puppet install script

apt-get install puppet -y

if [ ! -d /etc/puppet/files ]
then
    echo "Creating puppet files folder"
    mkdir /etc/puppet/files
fi


# TODO move this to puppet
locale-gen en_US en_US.UTF-8

# edit /etc/default/puppet to set default module folder
echo "modulepath=/var/puppet/modules" >> /etc/puppet/puppet.conf

#**************** END SCRIPT **********************

Then run

puppet -v /var/puppet/manifests/devserver.pp
